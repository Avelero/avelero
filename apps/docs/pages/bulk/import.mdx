# bulk.import

Documentation for bulk.import endpoint.

## Endpoint

`bulk.import` (Mutation)

## Purpose

Bulk import products or other resources

## Input Parameters

None. Uses authenticated user context from session.

## Return Value

```typescript
// Return type definition
{ success: boolean }
```

## Examples

```typescript
const result = await trpc.bulk.import.mutate({})
    ```

### Response

```json
{
  "success": true
}
```

## Error Handling

- `UNAUTHORIZED`: User is not authenticated
- `BAD_REQUEST`: Invalid parameters
- `INTERNAL_SERVER_ERROR`: Server error
