# bulk.update

Documentation for bulk.update endpoint.

## Endpoint

`bulk.update` (Mutation)

## Purpose

Bulk update multiple records

## Input Parameters

None. Uses authenticated user context from session.

## Return Value

```typescript
// Return type definition
{ success: boolean }
```

## Examples

```typescript
const result = await trpc.bulk.update.mutate({})
    ```

### Response

```json
{
  "success": true
}
```

## Error Handling

- `UNAUTHORIZED`: User is not authenticated
- `BAD_REQUEST`: Invalid parameters
- `INTERNAL_SERVER_ERROR`: Server error
