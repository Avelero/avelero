# passports.get

Documentation for passports.get endpoint.

## Endpoint

`passports.get` (Query)

## Purpose

Get a single passport by UPID

## Input Parameters

None. Uses authenticated user context from session.

## Return Value

```typescript
// Return type definition
{ success: boolean }
```

## Examples

```typescript
const result = await trpc.passports.get.query({})
    ```

### Response

```json
{
  "success": true
}
```

## Error Handling

- `UNAUTHORIZED`: User is not authenticated
- `BAD_REQUEST`: Invalid parameters
- `INTERNAL_SERVER_ERROR`: Server error
