# passports.update

Documentation for passports.update endpoint.

## Endpoint

`passports.update` (Mutation)

## Purpose

Update passport status or template

## Input Parameters

None. Uses authenticated user context from session.

## Return Value

```typescript
// Return type definition
{ success: boolean }
```

## Examples

```typescript
const result = await trpc.passports.update.mutate({})
    ```

### Response

```json
{
  "success": true
}
```

## Error Handling

- `UNAUTHORIZED`: User is not authenticated
- `BAD_REQUEST`: Invalid parameters
- `INTERNAL_SERVER_ERROR`: Server error
